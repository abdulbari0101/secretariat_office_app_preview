# secretariat_office_app
Developed a full-featured Parcel Management Mobile Application using Flutter &amp; GetX, featuring consignment tracking, driver management, financial accounting (double-entry bookkeeping), SMS notifications, PDF report generation, and SQLite database with backup/restore functionality.
